# KIM_load_ANSI
## How to load colorful ANSI pictures with your KIM

Convert an ANSI picture to Bytecode using Hans Ottens 8-bit Converter.

add 
pic_start: 
and 
pic_finish:

to the Picture File and compile everything with 64tass..
